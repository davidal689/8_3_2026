# Trang web Yêu Thương 💕

## Mở khóa
- **Mật khẩu mặc định:** `1234`
- Đổi mật khẩu trong file `script.js`: sửa dòng `const CORRECT_PASSWORD = '1234';`

## Cách chạy
Mở file `index.html` bằng trình duyệt (double-click hoặc kéo vào Chrome/Edge).

## Tùy chỉnh

### Music
- Trong `script.js`, mảng `songs` có thể thêm/sửa bài: `title`, `src` (link MP3), `cover` (link ảnh bìa).
- Double-click vào ảnh bìa trong player để dán URL ảnh mới.

### Image
- Trong `script.js`, mảng `imageUrls` có thể thay bằng link ảnh của bạn.

### Letter
- Trong `script.js`, sửa biến `fullText` để đổi nội dung thư.
- Tốc độ: 50 chữ/phút (sửa `CHARS_PER_MINUTE` nếu cần).

### Gift
- Nội dung "Yêu em" nằm trong `index.html` (class `gift-message`) và có thể sửa tùy ý.
